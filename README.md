# Codex-Digital-Marketing-Manager

JobSpec Studio is a lightweight browser app that turns a raw job description into a structured role brief.

## What it does

- extracts the likely role title, salary, location, working style, and experience signal
- separates responsibilities from requirements using lightweight text heuristics
- highlights keywords and likely stakeholders
- generates interview prompts and application talking points
- shows a structured JSON version of the analysis

## Run it

Open `index.html` in a browser.

No install step is required.
